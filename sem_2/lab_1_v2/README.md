# CLI-планировщик свадьбы (Вариант 72)

## Описание

Этот проект моделирует предметную область организации свадебного торжества.  
Система включает обязательные сущности:

- молодожены
- гости
- платье невесты
- кольца
- церемония
- банкет

Реализованные операции:

- выбор даты и мест проведения свадьбы
- выбор свадебного наряда
- приготовление свадебного меню
- организация церемонии
- подготовка к фотосессии

## Проектирование с учетом SOLID

- **S (Single Responsibility):**
  - `entities.py` содержит только структуры доменных данных.
  - `operations.py` определяет только бизнес-операции.
  - `storage.py` отвечает только за сохранение и загрузку.
  - `cli.py` отвечает только за взаимодействие через командную строку.
- **O (Open/Closed):**
  - Новые операции можно добавлять через реализацию интерфейса `PlanOperation`.
- **L (Liskov Substitution):**
  - Любая реализация `PlanRepository` может заменять другую.
- **I (Interface Segregation):**
  - Используются небольшие специализированные интерфейсы (`PlanOperation`, `PlanRepository`).
- **D (Dependency Inversion):**
  - `WeddingPlannerService` зависит от абстракции `PlanRepository`.

## Требования

- Python 3.11+
- `pytest` для запуска тестов

## Установка и запуск

```bash
python -m venv .venv
source .venv/bin/activate
pip install -U pip pytest
python main.py
```

## Сохранение состояния

Состояние программы сохраняется между запусками в файл:

- `data/wedding_plan.json`

## Тестирование

Запуск unit-тестов:

```bash
pytest -q
```
