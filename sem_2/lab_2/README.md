# Лабораторная работа 2 (Вариант 13)

Оконное приложение на Python (`tkinter`) для управления массивом записей о поездах.

## Запуск

```bash
python3 main.py
```

Опционально для календаря в диалоге добавления даты:

```bash
pip install tkcalendar
```

## Возможности

- MVC-архитектура с разделением на `model`, `view`, `controller`, `service`, `repository`.
- Меню и дублирующая панель инструментов.
- Диалог добавления записи.
- Отдельный диалог поиска с выводом результатов в этом же окне и пагинацией.
- Отдельный диалог удаления по условиям поиска.
- Хранение записей в SQLite базе данных.
- Сохранение в XML (DOM) и загрузка из XML (SAX).
- Сохранение/загрузка текущего массива в/из выбранной пользователем SQLite БД.
- Два режима отображения в главном окне: таблица и дерево.
- Постраничный вывод в главном окне и окне поиска.

## Генерация демонстрационных файлов

```bash
python3 generate_demo_files.py
```

Скрипт создаст 3 XML-файла с 50 уникальными осмысленными записями каждый в директории `demo_files`.
